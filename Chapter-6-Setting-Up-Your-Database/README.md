# Capítulo 6 — Setting Up Your Database

Criar e conectar um banco de dados **PostgreSQL** ao app e populá-lo (_seed_) com dados iniciais.

## 🎯 O que se aprende

- Criar um banco PostgreSQL (ex.: via Vercel/Neon)
- Conectar o projeto ao banco usando variáveis de ambiente (`.env`)
- Fazer o _seed_ do banco com dados iniciais
- Executar uma primeira query para validar a conexão
